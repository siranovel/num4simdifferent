## install ##

From rubygems:  
~~~
    [sudo] gem install num4simdiff
~~~

or from the git repository on github:  
~~~
    git clone https://github.com/siranovel/num4simdiffrent.git  
    cd num4simdiffrent  
    jruby -S gem build *.gemspec
    jruby -S gem install num4simdiff
~~~
